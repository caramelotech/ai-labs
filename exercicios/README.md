## Como usar

1. Leia o enunciado do exercício
2. Tente resolver por conta própria antes de buscar ajuda
3. Consulte as anotações em `/notes` se precisar revisar algum conceito
4. Compare sua solução com a dos colegas

## Exercício 01 - Identificando IA no Dia a Dia

**Objetivo:** Reconhecer aplicações de IA em produtos e serviços do cotidiano.

**Instrução:**
Liste ao menos **5 produtos ou serviços** que você usa regularmente e que utilizam Inteligência Artificial. Para cada um, descreva:

1. O nome do produto/serviço
2. Como a IA é usada nele
3. Qual tipo de IA você acredita que é utilizado (recomendação, reconhecimento de imagem, NLP, etc.)

**Critérios de sucesso:**

- [ ] Pelo menos 5 exemplos identificados
- [ ] Cada exemplo tem o tipo de IA descrito
- [ ] Os exemplos são de categorias diferentes (não todos do mesmo segmento)

## Exercício 02 - Criando seu Primeiro Prompt

**Objetivo:** Praticar a escrita de prompts eficazes usando o framework aprendido.

**Instrução:**
Escolha uma tarefa do seu trabalho ou estudos que poderia ser auxiliada por IA. Escreva um prompt completo usando o framework: **Papel + Instrução + Contexto + Exemplo**.

Exemplo de tarefa: "Resumir um artigo técnico para não-técnicos."

**Critérios de sucesso:**

- [ ] O prompt define claramente o papel da IA
- [ ] A instrução é específica e objetiva
- [ ] O contexto fornece informações relevantes
- [ ] O exemplo (se aplicável) demonstra o formato esperado
- [ ] O prompt foi testado em algum modelo de IA
