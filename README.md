---

## 🎯 Sobre Este Projeto

Este repositório contém uma curadoria completa de conteúdos sobre IA, focado em:

- **Conceitos Fundamentais de IA:** Entenda desde o básico até conceitos avançados
- **Prompt Engineering:** Aprenda a arte de conversar efetivamente com IAs
- **Recursos Curados:** Cursos, livros, ferramentas e materiais selecionados
- **Exemplos Práticos:** Aplicações reais e exercícios hands-on

## 🤝 Como Contribuir

Contribuições são muito bem-vindas! Este é um projeto comunitário e sua participação é fundamental.

### Formas de Contribuir

1. **Adicionar Conteúdo**
   - Use o [template](.github/TEMPLATE.md) para criar novos artigos
   - Siga a estrutura de pastas existente
   - Mantenha a linguagem clara e acessível

2. **Sugerir Recursos**
   - Adicione cursos, livros ou ferramentas úteis
   - Inclua descrição, nível e link

3. **Corrigir Erros**
   - Encontrou um erro? Abra uma issue ou faça um PR

4. **Melhorar Explicações**
   - Sugira formas mais claras de explicar conceitos
   - Adicione exemplos práticos

### Processo de Contribuição

1. Fork este repositório
2. Crie uma branch para sua feature (`git checkout -b feature/NovoConteudo`)
3. Commit suas mudanças (`git commit -m 'Adiciona novo conteúdo sobre X'`)
4. Push para a branch (`git push origin feature/NovoConteudo`)
5. Abra um Pull Request
